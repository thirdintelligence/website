# Third i — Official Website

> **[thirdi.net](https://thirdi.net)** — The public face of Third i, an AI-native creative marketing agency.

## About

Third i delivers film, web, and marketing at the speed of thought — powered by a workflow that gets smarter with every project.

This repository contains the source code for the official Third i website, built as a proof of concept for the agency's AI-native design and development capabilities.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** TailwindCSS 3+
- **Language:** TypeScript (strict mode)
- **Components:** shadcn/ui + custom glass-morphism system
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Netlify (auto-deploy on push to `main`)
- **Domain:** [thirdi.net](https://thirdi.net)

## Development

```bash
npm install
npm run dev
```

## Deployment

Continuous deployment via Netlify. Pushing to `main` auto-deploys to [thirdi.net](https://thirdi.net).
